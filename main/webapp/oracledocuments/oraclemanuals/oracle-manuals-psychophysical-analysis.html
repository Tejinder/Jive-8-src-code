Oracle Psychophysical Analysis

Not Mandatory, but if method is to be conducted, it should be applied as described in this database.

Overview

ORACLE Psychophysical Analysis method (PPA) is based on the evaluation of several products with varying levels of one attribute to assess ideal attribute levels. It provides actionable data for product development/manufacturing quality control, since consumer requirements can be linked to absolute levels of the attribute in question.

Psychophysical Analysis (PPA) is one of the two methods of Product Physical Evaluation alongside Consumer Physical Quality Evaluation (CPQE). Please see the attached document for further Methodology information.

Objectives

The method provides:

  • Consumer ideal level requirement for each product quality attribute.
  • Consumer acceptable range of variability around the ideal point (tolerance range).
  • Quantification of potential loss of consumer franchise in the event of a product attribute falling outside of the tolerance range to any given degree.
